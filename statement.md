PROBLEM STATEMENT:

In the digital era, food delivery applications have become increasingly popular due to convenience and time-saving features. 
This project aims to design a simple, user-friendly console-based Food Delivery System using Python.

The system allows users to:
* Log in to the application
* View a menu
* Place an order
* Calculate bill
* Apply discounts
* Track delivery status

The project is modularized into separate Python files to improve readability, maintainability, and scalability.

SCOPE OF THE PROJECT:

The online food delivery system aims to simplify the process of ordering food for all ages. 
It is a platform where customers can browse menus, place orders, track their purchases
This project covers the scope of user management,menu management, order processing, billing system, data storage

TARGET USERS:

This project aims to target customers of all age especially students and working professionals and also restaurant staffs and small food businesses
