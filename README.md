# ONLINE-FOOD-DELIVERY-SYSTEM
PROJECT TITLE:
ONLINE FOOD DELIVERY SYSTEM

OVERVIEW OF PROJECT:

In the digital era, food delivery applications have become increasingly popular due to convenience and time-saving features. This project aims to design a simple, user-friendly console-based Food Delivery System using Python.

FEATURES:

The system allows users to:
* Log in to the application
* View a menu
* Place an order
* Calculate bill
* Apply discounts
* Track delivery status

TECHNOLOGIES USED:

Programming Language: Python 3.x

Libraries Used: time

IDE/Environment: VS Code
